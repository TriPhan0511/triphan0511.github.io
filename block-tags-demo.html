Some common block tags

Container: article, aside, section, main

article tag - Definition and Usage

The article tag specifies independent, self-contained content.

An article should make sense on its own and it should be posible to distribute from the rest of the site.

Potential sources for the article lement:

  • Forum post
  • Blog post
  • News story
  • Comment

Example:

Google Chrome

Google Chrome is a free, open-source web browser developed by Google, released in 2008.

aside tag - Definition and Usage

The aside tag defines some content aside from the content it is placed in. The aside content should be related to the surrounding content.

Example:

My family and I visited The Epcot center this summer.

Epcot Center

The Epcot Center is a theme park in Disney World, Florida

section tag - Definition and Usage

The section tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.

Example: A section in a document, explaining what WWF is:

WWF

The World Wide Fund for Nature (WWF) is...

main tag - Definition and Usage

The main tag specifies the main content of a document. The content inside the main element should be unique to the document. It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search form.
Note: There must not be more than one main element in a document. The main element must NOT be a descendant of an article, aside, footer, header, or nav element.

Example:

Web Browsers

Google Chrome, Firefox, and Internet Explorer are the most used browsers today.

Google Chrome

Google Chrome is a free, open-source web browser developed by Google, released in 2008.

Internet Explorer

Internet Explorer is a free web browser from Microsoft, released in 1995.

Mozilla Firefox

Firefox is a free, open-source web browser from Mozilla, released in 2004.
